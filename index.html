Contrast, Alignment, Repetition, Proximity Task (CARP)

Bristol Balloon Festival 2023 - First Design

For my first design I used a background image where theres numerous hot airballoons displayed which depicts a repetition effect. In the centre of the poster I added the main title text "Bristol international fiesta" where I replaced the letter "o" from the word 'Bristol' with a vector hot air balloon.

Then under the main title I added all the other details that where needed for the event with first the text highlighting that it is 'Europes largest ballooning event' with underlining 'largest' to get that message across to the viewer.

The colours I chose where where a light yellow and white, with manipulating the opacity to create an effect where the text would blend with the background in some cases.

I seperated the content into 3 sections where the first section, at the very top, that displayed clearly the event date. The second section wsa the main title and other necessary information, and for the final section at the bottom of the poster I displayed all the sponsors of the event.

Bristol Balloon Festival 2023 - Second Design

For my second design I kept most elements from the first, but instead I used the events original logo as my centre text as I dived into research on their website where they used the same layout which would create a smooth link between website and poster.

I then replaced the top section instead of the date, with the text "Europes largest ballooning event" with some nice font.

Bristol Balloon Festival 2023 - Final Design

For my final and favourite design of the three is where I changed the background image with a beautiful huge hot air balloon in the centred and with multiple fireworks going on. This was a perfect choice for displaying repetition and proximity.

I chose to keep all the changes from my second design's layout with the logo centred and with the sponsors on the bottom as it creates a balanced all around visual to the viewer.

I felt that this design was the most appealing at a first glance and at the same time getting the message acroos and all the information that was needed for the viewer.

TYPOGRAPHY TASK

Axial System

This system shows that all elements are aligned to either the
right or left side of a central axis.

Bilateral System

Bilateral system is where All text is arranged symmetrically on
a single axis.

Dilational System

Dilatational system is where all elements expand from a central point in a circular fashion.

Grid System

The Grid system is a system of vertical and horizontal divisions

Modular System

The Modular system involves a series of nonobjective elements that are constructed as standardized units.

Radial System

In a radial system, the elements extend from a central point of focus.

Random System

In this system all elements appear to have no specific patter or relationship.

Transitional System

Transitional system is an informal system of layered banding
